# OIIA
spinning cat

## Installation:

    git clone https://github.com/ojazzz2505/OIIA.git
    cd OIIA

## Run the animation:
    python3 OIIA.py

exit with Ctrl+C


## Note: For best experience:

- Use a terminal with at least 120 columns width

- Run in fullscreen mode

- Ensure your terminal supports ANSI escape codes
